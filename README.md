# 🌍 Global AI Translator Bot & Mini App

Ye ek powerful **Telegram Bot** aur **Mini App** interface hai jo Gemini AI ka use karke instant translations provide karta hai.

---

### ⚡ Control Panel (Status)
- **Status:** 🟢 **ONLINE**
- **Hosting:** 🚀 **Cloud Optimized (Koyeb/Render)**
- **Model:** 🧠 **Gemini-Pro**

---

### 📁 Project Structure
- `main.py`: Core logic for Telegram bot and Flask backend.
- `logic.py`: Gemini AI setup and translation functions.
- `templates/index.html`: Web interface for the Mini App.
- `requirements.txt`: List of required libraries.

### 🌟 Key Features
- **Phonetic Pronunciation:** Sahi uchcharan batata hai.
- **Regional Support:** Hindi aur Gujarati language accurate results.
- **Global Languages:** Support for 100+ languages.
- **Zero-Cost:** Optimized for free hosting platforms.

### 🚀 Setup Instructions
1. Upload these files to your GitHub Repo.
2. Connect your GitHub Repo to Koyeb or Render.
3. Add your **Telegram Bot Token** in `main.py`.
4. Add your **Gemini API Key** in `logic.py`.
5. Trigger `/start` in your bot to test.
